# 📚 Hub Leitura - Testes Automatizados

Projeto de automação de testes E2E desenvolvido com Cypress para validar o formulário de contato do sistema Hub Leitura.

## 🚀 Tecnologias utilizadas

- Cypress
- JavaScript
- Node.js
- Git
- GitHub

## 📋 Cenários automatizados

- Envio do formulário com sucesso.
- Validação ao enviar sem preencher o nome.
- Validação ao enviar sem preencher o e-mail.
- Validação ao enviar sem selecionar o assunto.
- Validação ao enviar sem preencher a mensagem.

## ▶️ Como executar o projeto

1. Clone o repositório:

```bash
git clone https://github.com/laurajala/hub-leitura-teste-ui.git
```

2. Instale as dependências:

```bash
npm install
```

3. Execute os testes:

```bash
npx cypress open
```
